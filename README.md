# AI-Powered Pinterest Affiliate Automation System

An end-to-end AI automation workflow built using **n8n**, **Rainforest API**, **Gemini/OpenAI**, and **Google Sheets** to automate Pinterest affiliate content generation.

## Features

* Automated Amazon product sourcing
* AI-generated Pinterest titles & descriptions
* SEO-optimized hashtags and keywords
* Automated Google Sheets storage
* Scheduled workflow automation
* Real Amazon product integration
* Affiliate marketing content pipeline

---

# Tech Stack

* n8n
* Rainforest API
* Gemini / OpenAI API
* Google Sheets API
* JavaScript
* JSON Processing

---

# Workflow Architecture

Schedule Trigger
→ Product Category Rotation
→ Rainforest API Product Search
→ Data Cleaning & Processing
→ AI Content Generation
→ JSON Parsing
→ Google Sheets Automation

---

# What This Project Solves

This workflow automates repetitive affiliate marketing tasks by:

* sourcing real Amazon products
* generating Pinterest-ready marketing content
* organizing outputs automatically
* reducing manual content production time
